À propos

Application développée dans le cadre du cours Angular à l’EPF.

Le projet

Cette application permet de gérer une collection de films. Elle propose une interface moderne pour consulter, ajouter, modifier et supprimer des films.

L’objectif était de mettre en pratique les concepts fondamentaux d’Angular : composants, services, routing et gestion des données.

Technologies utilisées

  • Angular
  • TypeScript
  • Bootstrap
  • HTML / SCSS

Projet réalisé en 2026 – EPF
